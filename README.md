# HieroglyphicPatternAnalyzer

An interdisciplinary project for analysing reoccuring sequences and patterns in egyptian hieroglyphs.

## Structure

- **data/**: Raw data and processed data
- **src/**: Sourcecode for the database, restructure reading order, detection of patterns and web-app.
- **static/**: Static files for the web-app (CSS, JS, images)

## Installation

1. Clone Repository.
2. Install all dependencies `pip install -r requirements.txt`.
